# Marlin 3D Printer Firmware for my modified Anycubic i3 BQ_ZUM_MEGA_3D

# My hardware

  - Base mechanics: Anycubic i3 Mega version 4
  - Board: MKS Gen 1.4
  - Stepper Drivers: TMC 2130

## Wiring notes

### Stock Hardware
  - Z2 motor is connected to E1
  - Z2 endstop is connected to X+
